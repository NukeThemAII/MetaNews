# MetaNews

**Real‑time OSINT & Market Intelligence — signal only.**

MetaNews is an autonomous intelligence platform that monitors global events, markets, incidents, and technology in near real‑time, filters noise using AI, and delivers only what materially matters.

---

## Why MetaNews?

* Faster than mainstream media
* Quantified severity & confidence
* Built for investors, analysts, and operators

---

## Features

* Live OSINT ingestion (wars, disasters, incidents)
* Market‑moving alerts (stocks, crypto, commodities)
* AI‑scored events (severity / confidence)
* Free delayed feed + premium real‑time alerts
* Telegram bot integration

---

## Tech Highlights

* n8n Community Edition (automation brain)
* Gemini + GPT‑5.2 (analysis & verification)
* LangChain (multi‑step reasoning chains)
* PostgreSQL + pgvector (semantic search)
* Next.js + Tailwind (modern UI)

---

## Getting Started

1. Clone repo
2. Copy `.env.example`
3. `docker compose up -d`
4. Open n8n → import workflows

See **GEMINI.md** for the full build spec.
